YT-DLP GUI — портативная версия (Windows)

Кратко
- Назначение: удобная загрузка видео/аудио с сайтов, поддерживаемых yt-dlp.
- Платформа: Windows 10/11 x64.
- Установка: не требуется. Просто запустите .exe.
- Безопасность: приложение не устанавливается в систему и не отправляет телеметрию.

Первый запуск
- При первом запуске приложение скачает свежий движок yt-dlp (~18 МБ) в папку bin рядом с .exe.
- FFmpeg уже встроен в .exe и используется для склейки видео и аудио.
- В этой же папке создаются: config.json (настройки) и папка logs (журналы).

Важно
- Запускайте .exe из папки с правом записи (Документы/Загрузки/Рабочий стол). В Program Files может потребоваться админ-доступ.
- Смартскрин/антивирус может предупредить про неизвестного издателя (файл не подписан). Нажмите «Подробнее» → «Выполнить в любом случае», если скачивали с официальной ссылки.

Обновления
- Вкладка «Обслуживание» → «Обновить yt-dlp». Рекомендуется делать это регулярно.

Удаление
- Просто удалите .exe и рядом лежащие папки bin и logs, а также config.json — это всё.

Правовые замечания
- Используйте программу в рамках законодательства и условий сайтов. Загружайте только то, на что у вас есть права.

Поддержка
- Если что-то пошло не так, откройте папку logs и приложите файл журнала при обращении за помощью.
